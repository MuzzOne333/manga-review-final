Manga Review Website
A full-stack web application for manga enthusiasts to create, read, and manage manga reviews with user authentication.

Features
User Authentication - Register/Login with JWT security

Full CRUD Operations - Create, Read, Update, Delete reviews

Rating System - 1-10 star ratings for manga

Genre Tagging - Multiple genres per review

Responsive Design - Works on desktop & mobile

Real-time Updates - Reviews appear instantly after posting

| Backend    | Frontend   | Database | Auth   | Validation |
| ---------- | ---------- | -------- | ------ | ---------- |
| Node.js    | HTML5/CSS3 | MongoDB  | JWT    | Joi        |
| Express.js | Vanilla JS | Mongoose | bcrypt |            |
